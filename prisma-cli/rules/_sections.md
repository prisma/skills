# sections

Rule categories for `prisma-cli`.

| Section | Prefix | Description |
|---|---|---|
| Setup | `init` | Project initialization and bootstrap commands |
| Generation | `generate` | Prisma Client and generator output workflows |
| Development | `dev` | Local Prisma Postgres development runtime |
| Database | `db-` | Direct schema/data lifecycle commands |
| Migrations | `migrate-` | Development and production migration workflows |
| Utility | `studio`, `validate`, `format`, `debug` | Inspection, validation, formatting, and diagnostics |
