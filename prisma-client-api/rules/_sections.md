# sections

Rule categories for `prisma-client-api`.

| Section | Prefix | Description |
|---|---|---|
| Client Construction | `constructor` | PrismaClient setup and options |
| Model Queries | `model-queries` | CRUD operations for models |
| Query Shape | `query-options` | select/include/omit/ordering/pagination |
| Filtering | `filters` | Field and logical filtering operators |
| Relations | `relations` | Relation reads and nested writes |
| Transactions | `transactions` | Transaction patterns and options |
| Raw SQL | `raw-queries` | Typed and safe raw query usage |
| Client Methods | `client-methods` | `$connect`, `$disconnect`, `$extends`, events |
